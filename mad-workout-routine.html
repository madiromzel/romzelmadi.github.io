Why I Built It

I love working out, so I built a tool I’d actually use. Fast to set up routines, easy to tweak, and focused on execution. It also let me show off my React skills with state management, drag‑and‑drop, and a real‑time interval timer, all wrapped in a bold UI that matches the training mindset.
