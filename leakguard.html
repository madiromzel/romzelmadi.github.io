Key Features

Real-Time Leak Alerts

  • Live MQTT updates with clear leak status.
  • Instant visual indicators for critical events.
  • Low-latency telemetry for quick response.

Historical Analytics (Firebase)

  • Firestore logging for audit trails.
  • Trend charts for sensor values and leak events.
  • Timestamped records for reporting.

Device Control & Safety

  • Remote threshold tuning from the dashboard.
  • MQTT commands for buzzer and test mode.
  • On-device safety fallback when offline.

Unified Dashboard

  • Real-time + historical data in one view.
  • Streamlined cards for current status.
  • Fast filtering and readable charts.

Reliable Connectivity

  • Auto-reconnect for MQTT sessions.
  • Secure TLS connection to broker.
  • Resilient to Wi-Fi dropouts.

IoT End-to-End Flow

  • ESP8266 to MQTT for live data.
  • HTTP to Node.js to Firestore for persistence.
  • Vue + Pinia for reactive UI state.

Why I Built It

LeakGuard was built as a final project requirement for the Embedded Systems subject. Water leaks are often unnoticed until they cause damage and high repair costs, especially in areas that are rarely checked. The system uses a NodeMCU (ESP8266), water leak/moisture sensors (YL-69 or similar), a buzzer for local alerts, a power supply, jumper wires, a breadboard, and a power adapter to deliver real-time detection and instant alerts.
